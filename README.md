# Fraud-Detection-Dashboard
An end-to-end Real-Time Financial Fraud Detection System powered by XGBoost, SMOTE, and Explainable AI (SHAP), featuring a live Streamlit operations dashboard.
This directory includes a few sample datasets to get you started.
